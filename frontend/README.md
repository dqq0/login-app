# 🖥️ Deathcloud - Frontend Web

Interfaz de usuario para la plataforma de comunicaciones Deathcloud, construida para conectarse con nuestro servidor de WebSockets.

## 🛠️ Tecnologías
* HTML5, CSS3, JavaScript
* Cliente de Socket.io

## ⚙️ Ejecución

1. Asegúrese de que el servidor Backend esté corriendo primero en el puerto 3000.
2. Abra el archivo `index.html` utilizando la extensión **Live Server** (recomendado en VS Code) para evitar problemas de CORS.
3. Inicie sesión para comenzar a transmitir.
